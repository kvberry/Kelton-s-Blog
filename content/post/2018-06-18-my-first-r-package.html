For the iX Week 2 project, Will Shuhda and I created a R Package. The premise of our Package was to find the prime numbers from 1 to 1000. After considerable thought, we developed the a function which would return boolean value of true if the number was prime and false if it was not prime. After we pushed our package to R, we successfully had created our “Prime_Number” R Package.
